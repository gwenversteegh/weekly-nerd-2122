## Typescript

### What is typescript?
TypeScript adds additional syntax to JavaScript to support a tighter integration with your editor. Catch errors early in your editor. This is what Typescriptlang says about typescript. But what does it mean? When you write JS you always get some errors and sometimes you don't even know what they mean. Typescript helps you define these errors and help you prefent them. It gives you feedback on your code while you're writing it. Because of this, you see errors earlier, before the whole applications crashes down. TypeScript uses compile time type checking. Which means it checks if the specified types match before running the code, not while running the code.

## Cascade layers

### What are Cascade layers?
Cascading layers is a CSS feature that allows us to define explicit contained layers of specificity, so that we have full control over which styles take priority in a project without relying on specificity hacks or !important. This allows us to override styles from elsewhere in our code (or a third-party tool)
